bitcoinaddress.js

JavaScript library for human-friendly Bitcoin address presentation, payments and sending bitcoins.

View on Github

bitcoinaddress.js is a JavaScript library allowing you to easily mark existing bitcoin addresses on the HTML page and add send bitcoins, QR code and copy actions for them. bitcoinaddress.js uses Bitcoin URI scheme and you can send amount, address label and message for the transactions.

Bitcoin URI scheme supports making payment from many web, desktop and mobile wallets, including but not limited to, Armory, Electrum, blockchain.info, Coinbase, Multibit and naturally, Bitcoin-QT.

If you combine bitcoinaddress.js with bitcoinprices.js, you can nominate the amount of send payment in fiat currencies like USD and EUR, and bitcoinprices will convert the amounts to Bitcoins in-fly, using market exchange rates.
