bitcoinaddress.js

JavaScript library for human-friendly Bitcoin address presentation, payments and sending bitcoins

bitcoinaddress.js is a JavaScript library allowing you to easily mark existing bitcoin addresses on the HTML page and add send bitcoins and copy actions for them.
